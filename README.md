## ⏱ 1만 시간의 법칙 (the_tenthousand_hour_rule)


'1만 시간의 법칙'에 따라
한 분야의 전문가가 되기 위한 시간인 
10,000 시간을 채우는데 걸리는 시간을 계산해줍니다.

Following 'the 10,000 hour rules'
calculate the number of days you need to invest to become an expert in a field.
